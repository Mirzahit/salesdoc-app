Динамика компании по неделям

    
   

   

    Оплаты (T) — сумма всех оплат из «Доходы» в периоде.  
    Оплаты по новым клиентам (T) — статьи с «нов» (нов клиент / нов внедрение / нов интеграция), без абонентских.  
    Внедрения (шт) — число оплат с заполненной «Датой активации».  
    Лиды (шт) — из таблицы маркетинга (LEADS_SHEET, Лист1) + Meta Ads CSV (после последней даты в Листе).
